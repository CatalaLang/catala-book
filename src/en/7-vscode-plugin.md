## The VSCode plugin

[VSCode](https://code.visualstudio.com/) is text editor officially supported and
maintained by the Catala team, through the [Catala VS Code
extension](https://marketplace.visualstudio.com/items?itemName=catalalang.catala).
See the [installation guides](./1-1-0-installing.md) for instructions about how
to install Catala and the VSCode extension.

The extension provides out-of-the-box usual features that you'd expected from 
an IDE for programming in Catala. It also features custom interfaces for 
Catala-specific interactions like the [test case editor](./7-1-test-gui.md).
