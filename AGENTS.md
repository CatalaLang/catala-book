Always strive for simple language when translating.

For instance "Veuillez faire preuve de prudence lorsque vous envisagez d’automatiser" -> "Faites preuve de prudence" ou "Soyez prudent" (first
form is better as it is gender neutral)

Hunt unneeded weasel words / extra adjectives.

Whenever you encounter an instance of "the Catala team", change
the sentence to "we" (shorter and more idiomatic)

French Catala identifiers (scope and variable names) should have
accents when needed e.g. Décédé (not Decede), date_décès (not date_deces)

Try to adopt an idiomatic French tone e.g. "Félicitations pour avoir terminé le tutoriel Catala !" -> "Vous avez terminé le tutoriel Catala, félicitations !"

However, note that accuracy is more important than idiomatic sentences
as this is a technical book.
